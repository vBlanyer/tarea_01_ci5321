## Descripción general

Se implementa un triángulo equilátero renderizado en un canvas con los vértices coloreados en rojo, verde y azul sobre un fondo ligeramente oscuro. Para observar los distintos niveles de abstracción y cómo eso conecta con la cadena de renderizado.

## Ejecucion 

Para arrancar el programo es necesario ubicarse dentro de la carpeta con la API a correr. Puede ser la carpeta de webgl, twtl o three, una vez ubicado correr el siguiente comando para poder ejecutar un servidor y poder visualizar en el navegador.

```bash
npm run dev
```

Y colocar en el navegador el enlace
